## 앱 설명

### 짧은 설명 (80자 이내)
일산 YCITY 주차장에서 내 차량의 위치를 쉽고 빠르게 확인하는 앱

### 전체 설명 (4000자 이내)
🚗 **YCITY+ - 스마트 주차 위치 확인 서비스**

일산 YCITY 아파트 주민을 위한 맞춤형 주차 위치 확인 앱입니다. 복잡한 지하주차장에서도 내 차량을 쉽고 빠르게 찾을 수 있습니다.

**🎯 주요 기능**
• **실시간 차량 위치 확인**: 동, 호수, 시리얼번호만 입력하면 즉시 주차 위치 확인
• **정확한 층 정보**: B1, B2, B3, B4 등 정확한 주차층 정보 제공
• **간편한 사용법**: 직관적인 인터페이스로 누구나 쉽게 사용
• **안전한 데이터**: 개인정보 보호를 위한 보안 시스템 적용
• **빠른 응답**: 최적화된 네트워크 통신으로 빠른 정보 제공

**💡 사용 방법**
1. 앱 실행 후 동, 호수 입력
2. 차량 시리얼번호 입력
3. 조회 버튼 클릭
4. 실시간 주차 위치 확인

**🏢 YCITY+ 전용 서비스**
- 일산 YCITY 아파트 주민 전용
- 실제 주차 관리 시스템과 연동
- 정확하고 신뢰성 있는 정보 제공

**🔒 개인정보 보호**
- 최소한의 정보만 수집
- 안전한 HTTPS 통신
- 개인정보 암호화 처리
