개인정보처리방침

앱명: YCity Plus  |  시행일: 2026년 3월 29일

YCity Plus(이하 "앱")는 사용자의 개인정보를 중요시하며, 수집한 개인정보를 앱 기능 제공 목적 외에 활용하지 않습니다. 본 앱은 외부 서버로 개인정보를 전송하지 않으며, 모든 데이터는 사용자 기기 내에만 저장됩니다.

1. 수집하는 정보

앱은 주차 위치 조회 및 이력 관리 서비스 제공을 위해 아래 정보를 수집합니다.

  • 동·호수: 사용자가 직접 입력하는 거주 동/호 번호 (예: 103동 4705호)
  • 차량 시리얼 번호: 사용자가 직접 입력하는 차량 식별 번호
  • 주차 이력: 입차·출차 시각, 주차 층, 주차 시간 등 앱이 자동으로 기록하는 이용 내역

2. 정보 수집 방법

  • 사용자가 앱 설정 화면에서 직접 입력
  • 아파트 단지 주차 서버에서 주차 상태를 조회하여 자동 기록

3. 정보 이용 목적

  • 현재 차량 주차 위치(층) 조회
  • 입·출차 이력 자동 기록 및 통계 제공
  • 홈 화면 위젯에 최신 주차 정보 표시

4. 정보 저장 위치 및 보존 기간

수집된 모든 정보는 사용자 기기 내 로컬 저장소(SQLite)에만 저장됩니다. 외부 서버나 클라우드로 전송되지 않습니다.

  • 사용자 정보(동·호수, 시리얼 번호): 사용자가 삭제하기 전까지 보관
  • 주차 이력: 사용자가 삭제하기 전까지 보관
  • 앱 삭제 시 기기에 저장된 모든 데이터가 함께 삭제됨

5. 제3자 제공

앱은 수집된 개인정보를 외부 제3자에게 제공하지 않습니다. 다만, 주차 위치 조회를 위해 아파트 단지 내 주차 서버(로컬 네트워크)에 차량 시리얼 번호를 전송하여 주차 위치 정보를 요청합니다. 이 요청은 조회 목적으로만 사용되며 서버에 저장되지 않습니다.

6. 개인정보 보호

  • 앱은 인터넷 서버로 개인 식별 정보를 전송하지 않습니다
  • 기기 잠금 설정을 통해 데이터를 보호할 것을 권장합니다
  • 앱 삭제 시 모든 저장 데이터가 완전히 제거됩니다

7. 사용자의 권리

사용자는 언제든지 아래 방법으로 개인정보를 관리할 수 있습니다.

  • 조회·수정: 앱 설정 화면에서 직접 확인 및 변경
  • 삭제: 앱 내 이력 삭제 기능 또는 앱 삭제를 통해 모든 데이터 제거

8. 아동 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 권한 사용

  • 인터넷(INTERNET): 아파트 단지 주차 서버와 통신하여 주차 위치 조회
  • 네트워크 상태(ACCESS_NETWORK_STATE): 연결 상태 확인
  • 백그라운드 작업(RECEIVE_BOOT_COMPLETED, SCHEDULE_EXACT_ALARM): 홈 위젯 주기적 업데이트

위 권한들은 앱의 핵심 기능 제공을 위한 최소한의 범위로만 사용됩니다.

10. 개인정보처리방침 변경

본 방침이 변경될 경우 앱 업데이트를 통해 사용자에게 공지하며, 변경 사항은 시행일로부터 적용됩니다.
